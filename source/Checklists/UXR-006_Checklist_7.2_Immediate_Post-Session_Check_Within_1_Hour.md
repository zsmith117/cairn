# Immediate Post-Session Check (Within 1 Hour)

**Source:** UXR-006 SOP
**Section:** 7.2
**Version:** 1.0
**Date:** January 2026

---

**Post-Session Quality Checklist:**

```
IMMEDIATE POST-SESSION QUALITY CHECK (Within 1 hour)

SESSION LOGISTICS:
[ ] Recording saved and backed up
[ ] Session metadata complete (participant, date, duration)
[ ] Any technical issues documented

NOTE COMPLETENESS:
[ ] All major questions/topics documented
[ ] At least 5-10 key quotes captured
[ ] Behavioral observations included
[ ] Context provided for responses
[ ] Surprises and unexpected moments flagged
[ ] No major gaps in coverage

NOTE QUALITY:
[ ] Abbreviations expanded where needed for clarity
[ ] Bracketed clarifications added where helpful
[ ] Sections that need filling from recording marked
[ ] Speaker attribution clear (who said what)

SESSION SUMMARY:
[ ] Session summary completed (Template 5.4)
[ ] 3-5 key findings documented
[ ] Notable quotes listed
[ ] Follow-up questions noted
[ ] Methodological notes captured

ANONYMIZATION:
[ ] Participant identified by code, not real name
[ ] No other PII in notes (company names, colleague names)

STORAGE:
[ ] Files properly named per convention
[ ] Saved in correct study folder
[ ] Backed up in secure location

SHARING:
[ ] Session summary shared with team (if collaborative study)
[ ] Any urgent insights communicated immediately

PASS/FAIL: [ ] Pass - Ready for use  [ ] Needs work - [What's needed]
```
