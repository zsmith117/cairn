# 24-Hour Quality Check

**Source:** UXR-006 SOP
**Section:** 7.3
**Version:** 1.0
**Date:** January 2026

---

**24-Hour Checklist:**

```
24-HOUR POST-SESSION QUALITY CHECK

TRANSCRIPTION:
[ ] Transcription requested if needed
[ ] Or determined transcription not needed with rationale

NOTE FINALIZATION:
[ ] Notes reviewed for completeness
[ ] Missing information filled in from recording (if applicable)
[ ] Unclear sections clarified
[ ] Notes cleaned up for readability

COLLABORATION:
[ ] If multiple observers, notes consolidated
[ ] Team debrief completed
[ ] Additional observations from team incorporated

PEER REVIEW (if applicable):
[ ] Peer review completed
[ ] Feedback incorporated

READY FOR NEXT SESSION:
[ ] Insights from this session inform next session
[ ] Any guide adjustments made
[ ] Follow-up questions integrated into guide

PASS/FAIL: [ ] Pass - Documentation complete  [ ] Needs attention - [What]
```
