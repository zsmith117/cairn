# Theme Development Quality Check

**Source:** UXR-007 SOP
**Section:** 7.2
**Version:** 1.0
**Date:** January 2026

---

After developing themes:

| Check | Standard | Pass? |
|-------|----------|-------|
| **Distinctness** | Themes are clearly different from each other | [ ] |
| **Coherence** | Each theme is internally consistent | [ ] |
| **Evidence** | Each theme supported by multiple data points | [ ] |
| **Relevance** | Themes address research questions | [ ] |
| **Meaningfulness** | Themes reveal something important | [ ] |
| **Prevalence documented** | Frequency/distribution of each theme noted | [ ] |
| **Variations documented** | Differences across segments identified | [ ] |
| **Relationships mapped** | Connections between themes explained | [ ] |
