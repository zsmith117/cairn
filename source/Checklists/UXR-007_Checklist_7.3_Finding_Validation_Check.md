# Finding Validation Check

**Source:** UXR-007 SOP
**Section:** 7.3
**Version:** 1.0
**Date:** January 2026

---

Before finalizing findings:

```
FINDING VALIDATION CHECKLIST

For each major finding:

EVIDENCE STRENGTH:
[ ] Supported by multiple participants ([N] of [Total])
[ ] Consistent expression across participants
[ ] Multiple data points (quotes + behaviors + context)
[ ] Direct evidence available (not inferred)

VALIDITY:
[ ] Negative case analysis conducted
[ ] Contradictory evidence considered
[ ] Alternative interpretations evaluated
[ ] Bias check performed

QUALITY:
[ ] Finding is clear and specific
[ ] Finding is meaningful (not obvious)
[ ] Finding addresses research question
[ ] Finding is defensible with evidence

READINESS:
[ ] Can trace finding back to raw data
[ ] Have quotes to support finding
[ ] Can explain to stakeholders
[ ] Ready to withstand challenge

ALL CHECKS PASSED: [ ] Yes [ ] No - Needs: _____________
```
