# Pre-Communication Quality Check

**Source:** UXR-008 SOP
**Section:** 7.1
**Version:** 1.0
**Date:** January 2026

---

Before presenting or distributing:

```
PRE-COMMUNICATION CHECKLIST

CONTENT QUALITY:
[ ] All findings grounded in evidence
[ ] Prevalence/frequency clearly stated
[ ] Quotes are verbatim and attributed
[ ] Context and limitations clearly stated
[ ] Insights go beyond observations (explain "why")
[ ] Recommendations are specific and actionable

NARRATIVE QUALITY:
[ ] Clear story arc from context to recommendations
[ ] Key messages identified and emphasized
[ ] "So what?" is clear for each finding
[ ] Appropriate level of detail for audience
[ ] Compelling and memorable

VISUAL QUALITY:
[ ] Slides/report are visually clear
[ ] Text is readable (font size, contrast)
[ ] Visuals support points (not decorative)
[ ] Consistent formatting throughout
[ ] Professional appearance

ACCURACY:
[ ] Numbers and statistics verified
[ ] Participant IDs correct
[ ] Methodology accurately described
[ ] No typos or errors
[ ] All links work

COMPLETENESS:
[ ] All research questions addressed
[ ] Key findings included
[ ] Recommendations provided
[ ] Next steps clear
[ ] Contact information included

PEER REVIEW:
[ ] Reviewed by colleague or Research Lead
[ ] Feedback incorporated
[ ] Approved for distribution

READY TO SHARE: [ ] Yes [ ] No - Needs: _______________
```
