# Post-Communication Quality Check

**Source:** UXR-008 SOP
**Section:** 7.3
**Version:** 1.0
**Date:** January 2026

---

Within 1 week after communication:

```
POST-COMMUNICATION ASSESSMENT

DELIVERY:
[ ] Communication happened as planned
[ ] Key messages were delivered
[ ] Audience engaged appropriately
[ ] Questions were answered
[ ] Materials distributed

STAKEHOLDER RESPONSE:
[ ] Stakeholders understood findings
[ ] Stakeholders accepted findings (or understood their concerns)
[ ] Discussion was productive
[ ] Clear next steps established
[ ] Positive feedback received

FOLLOW-UP:
[ ] Summary email sent
[ ] Materials shared
[ ] Action items documented
[ ] Owners assigned
[ ] Timeline established

ISSUES TO ADDRESS:
[ ] No major misunderstandings
[ ] No research misuse observed
[ ] No significant gaps in communication

If issues identified: [Action plan to address]

LESSONS LEARNED:
What worked well: _____________________
What to improve next time: _____________________
```
