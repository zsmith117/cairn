# Affinity Mapping Facilitation Guide

**Source:** UXR-007 SOP
**Section:** 5.2
**Version:** 1.0
**Date:** January 2026

---

```
AFFINITY MAPPING SESSION - FACILITATION GUIDE

===================================================================
STUDY: [Name]
DATE: [Date]
FACILITATOR: [Name]
SCRIBE: [Name]
PARTICIPANTS: [Names]
===================================================================

PRE-SESSION SETUP:

MATERIALS:
[ ] Sticky notes (3-4 colors)
[ ] Sharpies (1 per person)
[ ] Large wall space or digital board ready
[ ] Printed session notes available
[ ] Timer
[ ] Snacks and water
[ ] Camera for documentation

ROOM SETUP:
[ ] Wall space cleared
[ ] Tables for writing
[ ] Notes accessible
[ ] Breaks planned

===================================================================

FACILITATOR SCRIPT:

1. INTRODUCTION (5 min)

"Welcome everyone. Today we're going to analyze the research from
[Study name] using affinity mapping. This is a collaborative process
where we'll organize observations into themes.

Here's what we're going to do:
[Explain process - show slide if helpful]

Ground rules:
- One observation per sticky note
- Use participant quotes when possible
- Stay grounded in data
- Silent grouping before discussion
- Anyone can move any note
- No judgment - all observations are valid

Questions before we start?"

---

2. DATA REVIEW (10 min)

"Take 10 minutes to review the data. Refresh your memory on what
participants said and did. You can review session notes, summaries,
or transcripts. Make mental note of key observations."

[Set timer for 10 minutes]

---

3. OBSERVATION CAPTURE (45-60 min)

"Now, write observations on sticky notes. One observation per note.

Focus on:
- What participants said (use quotes)
- What participants did (behaviors)
- Pain points
- Needs
- Surprises
- Contradictions

Write clearly so others can read. Aim for at least 50 notes.

You have 45 minutes. I'll give a warning at 30 minutes."

[Set timer; circulate to answer questions]

[At 30 min: "15 minutes remaining"]
[At 40 min: "5 minutes - finish your current thought"]

---

4. SILENT POSTING (15 min)

"Now we'll post notes on the wall. Do this silently - no talking yet.

As you post, read others' notes. Start thinking about which notes
are related.

Begin!"

[Set timer; monitor process; answer questions via gesture/whisper]

---

5. BREAK (10 min)

"Let's take a 10-minute break. When we return, we'll group the notes."

---

6. SILENT GROUPING (30-45 min)

"Now we'll group related observations together. This is still silent.

Rules:
- Move notes that seem related close together
- Anyone can move anyone's note
- If someone moves a note you don't agree with, you can move it back
- It's OK if notes move multiple times
- Trust the process
- If a note doesn't fit anywhere, leave it alone for now

Begin!"

[Set timer; observe process; intervene only if stuck]

[Watch for:
- Groups forming
- Notes being moved repeatedly (normal)
- Confusion (rare - usually resolves)
- Completion (grouping stabilizes)]

[When grouping stabilizes or time reached: "Let's pause here"]

---

7. GROUP DISCUSSION & LABELING (45-60 min)

"Now we can talk! Let's look at these groups and name them.

Starting with this group [point to one], what theme unifies these
observations?"

[Facilitate discussion:]
- "What ties these together?"
- "What would we call this theme?"
- "Does this note really belong here?"
- "Is this theme distinct from that one?"

[For each group:
- Discuss what unifies it
- Name the theme
- Write theme name on different color note
- Challenge: Does everything fit?
- Consider: Sub-themes needed?]

[Look for super-groups:]
"Are there themes of themes? Do some groups relate?"

[Create hierarchy if helpful]

---

8. REFINEMENT (20-30 min)

"Let's step back and look at the whole map.

Questions:
- Are themes distinct?
- Do any overlap too much?
- Are there gaps - something missing?
- Any orphan notes we need to address?
- What's most important?
- What relationships do we see?"

[Facilitate refinement discussion]

[Optional: Dot voting for priority themes]
"Everyone gets 3 dots. Place them on themes you think are most important."

---

9. DOCUMENTATION (15 min)

[Scribe activities:]
[ ] Photograph final map (multiple angles)
[ ] List themes and sub-themes
[ ] Note key decisions made
[ ] Document questions raised
[ ] Capture voting results if applicable

[Team activity:]
"Let's each share one surprise or key takeaway from this process."

[Go around room]

---

10. CLOSING (5 min)

"Thank you all. This was excellent collaboration.

Next steps:
- [Scribe name] will document this digitally
- [Analysis lead] will develop these into formal findings
- We'll share results by [date]
- Questions or concerns, reach out to [contact]

Thanks again!"

===================================================================

POST-SESSION:

[ ] Transfer photos to project folder
[ ] Document themes in digital format
[ ] Distribute notes to team
[ ] Schedule follow-up if needed
[ ] Thank participants

===================================================================

FACILITATION TIPS:

STUCK IN SILENT GROUPING:
- Let it play out; discomfort is normal
- If truly stuck after 10 min, allow brief clarifying questions
- Resist urge to tell people where notes go

DOMINATING VOICES IN DISCUSSION:
- "Let's hear from someone who hasn't spoken yet"
- "Hold that thought - other perspectives?"
- Direct questions to quiet participants

DISAGREEMENT ON GROUPING:
- "Both views have merit. Let's try it both ways."
- "Where's the evidence? What do the notes say?"
- "Can we create a sub-theme to capture both?"

RUNNING LONG:
- Skip or shorten break
- Set stricter time limits on discussion
- Focus on top priority themes
- Schedule follow-up session for remaining themes

===================================================================
```
