EVIDENCE STRENGTH:

1. FREQUENCY / PREVALENCE

How many participants/sessions showed this pattern?
[N] of [Total] = [%]

Assessment:
[ ] Strong (>75% of participants)
[ ] Moderate (50-75%)
[ ] Emerging (25-50%)
[ ] Weak (<25%)

2. CONSISTENCY

How consistent is the pattern across participants?
[ ] Very consistent - expressed similarly by all
[ ] Mostly consistent - core similarity, some variation
[ ] Moderate consistency - recognizable pattern with variation
[ ] Inconsistent - widely varied expressions

3. INTENSITY / IMPACT

How significant is this for participants?
[ ] High - Strongly emphasized; clear importance
[ ] Medium - Notable but not primary focus
[ ] Low - Mentioned but not emphasized

Evidence of intensity:
- Emotional response: [Description]
- Time spent discussing: [Amount]
- Unsolicited mentions: [Y/N]
- Rated as high priority by participant: [Y/N]

===================================================================

EVIDENCE QUALITY:

4. DIRECT EVIDENCE

[ ] Have verbatim quotes supporting this finding (minimum 3)
[ ] Have behavioral observations supporting this
[ ] Have contextual data explaining this pattern
[ ] Multiple data sources support this (notes + transcript + observation)

5. CONTEXT

[ ] Context is preserved in interpretation
[ ] Conditions under which this occurs are documented
[ ] Understand why this pattern exists (not just that it exists)

===================================================================

VALIDITY CHECKS:

6. NEGATIVE CASE ANALYSIS

Actively searched for counter-examples?
[ ] Yes [ ] No

Found any contradictory data?
[ ] No contradictions found
[ ] Minor contradictions found - do not invalidate finding
[ ] Significant contradictions found - finding needs revision

If contradictions found:
[Description of contradictory evidence]

How addressed:
[ ] Refined finding to accommodate
[ ] Identified boundary conditions
[ ] Revised interpretation

7. BIAS CHECK

Confirmation bias check:
[ ] This finding contradicts my expectations (less bias risk)
[ ] This finding confirms my expectations (examined extra carefully)

Did I expect to find this before analyzing data?
[ ] No [ ] Yes - If yes, extra scrutiny applied: [How]

Would I believe this if contradicted my preferences?
[ ] Yes [ ] Unsure - reexamined

8. ALTERNATIVE INTERPRETATIONS

Considered other ways to interpret this data?
[ ] Yes [ ] No

Alternative interpretation:
[Description]

Why primary interpretation is preferred:
[Rationale]

===================================================================

RESEARCH QUESTION ALIGNMENT:

9. RELEVANCE

Does this finding address research questions?
[ ] Directly addresses primary research question
[ ] Addresses secondary research question
[ ] Interesting but tangential - document but deprioritize
[ ] Off-topic - exclude from primary findings

Which research question: [Specify]

===================================================================

ACTIONABILITY:

10. CLARITY

Is the finding clear and specific?
[ ] Yes - Clear what we learned
[ ] Somewhat - Needs clarification
[ ] No - Too vague or ambiguous

If unclear, what needs clarification:
[Description]

11. MEANINGFULNESS

Is this finding meaningful and non-obvious?
[ ] Yes - Reveals something new and important
[ ] Somewhat - Some value but limited novelty
[ ] No - States the obvious

Why this matters:
[Explanation of significance]

===================================================================

DEFENSIBILITY:

12. TRACEABILITY

Can I trace this finding back to raw data?
[ ] Yes - Have specific quotes and observations
[ ] Partially - Have some evidence but gaps
[ ] No - Would struggle to defend

Evidence location:
[Reference to where evidence is documented]

13. STAKEHOLDER CHALLENGE READINESS

If stakeholder said "I don't believe that," could I:
[ ] Show multiple quotes supporting it
[ ] Explain the pattern with specific examples
[ ] Demonstrate consistency across participants
[ ] Explain contradictory data and why finding still holds

===================================================================

STRENGTH OF FINDING:
[ ] Strong - Well-supported, consistent, validated
[ ] Moderate - Supported but some limitations
[ ] Weak - Limited support or significant issues

[ ] High confidence - Ready to communicate

[ ] Low confidence - Needs more work or more data

[ ] Yes - Validated and ready

[ ] Revisit interpretation
