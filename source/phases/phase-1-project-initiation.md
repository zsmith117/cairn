# Phase 1: Project Initiation

Getting your research project started with proper alignment, approvals, and setup.

---

## Phase Overview

**Objective:** Establish a solid foundation for your research project through stakeholder alignment, ethics compliance, and proper project setup.

**Duration:** Typically 1–2 weeks depending on approval processes.

**Key Outcomes:**

- Stakeholder alignment on research objectives
- Ethics approval obtained (if required)
- Consent procedures established
- Project tracking set up in Research Project Tracker

---

## Step-by-Step Process

### 1. Stakeholder Alignment

Before any research begins, ensure alignment with stakeholders on:

- Research questions and objectives
- Scope and constraints
- Timeline expectations
- How findings will be used

> Reference [UXR-004: Research Planning & Scoping](#sop/UXR-004/procedure) for detailed guidance on stakeholder alignment procedures.

### 2. Ethics & Consent Setup

Establish your ethics and consent framework:

- Determine if IRB/ethics review is required
- Prepare consent forms appropriate to your method
- Set up consent collection process
- Brief team on consent procedures

> Reference [UXR-001: Informed Consent & Ethics](#sop/UXR-001/procedure) for all consent requirements and templates.

### 3. Project Setup

Create your project infrastructure:

1. Add project to **Research Project Tracker**
2. Set `Current Phase` to `1-Initiation`
3. Complete project metadata (Owner, Summary, Timeline)
4. Create project folder structure for documentation
5. Schedule kickoff meeting if needed

## Phase Completion Checklist

Before moving to Phase 2, confirm:

- [ ] Research objectives documented and approved by stakeholders
- [ ] Ethics requirements identified and approval obtained (if needed)
- [ ] Consent forms prepared and process established
- [ ] Project added to Research Project Tracker
- [ ] Project documentation structure created

---

## What's Next

Once Phase 1 is complete:

1. Update your project's `Current Phase` to `2-Design`
2. Add `P1-Complete` to `Phase Progress`
3. Proceed to **[Phase 2: Research Design →](#guide/phase-2)**
