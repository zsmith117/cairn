# Phase 3: Field Work

Executing your research — recruitment, data collection, and session documentation.

---

## Phase Overview

**Objective:** Successfully execute your research plan by recruiting participants, conducting sessions, and maintaining thorough documentation.

**Duration:** Varies by study — typically 2–4 weeks for qualitative studies.

**Key Outcomes:**

- Target number of participants recruited and scheduled
- All sessions conducted per protocol
- Complete session documentation
- Data securely stored per compliance requirements

---

## Step-by-Step Process

### 1. Recruitment Execution

Launch and manage recruitment:

- Activate recruitment channels
- Screen incoming participants
- Schedule qualified participants
- Manage participant communications
- Track recruitment progress

> Reference [UXR-003: Participant Recruitment & Screening](#sop/UXR-003/procedure) for detailed recruitment execution procedures.

### 2. Session Preparation

Before each session:

- Review participant profile
- Prepare recording equipment
- Confirm logistics with participant
- Brief any observers/note-takers
- Have consent process ready

### 3. Session Execution

During sessions:

- Obtain informed consent
- Follow interview/moderation guide
- Maintain appropriate documentation
- Handle any participant concerns appropriately

> Reference [UXR-001: Informed Consent & Ethics](#sop/UXR-001/procedure) for consent procedures and participant rights.

### 4. Documentation

After each session:

- Complete session notes
- Secure recordings and data
- Update tracking
- Note any immediate insights or issues

> Reference [UXR-006: Note-Taking & Documentation](#sop/UXR-006/procedure) for documentation standards.

### 5. Data Management

Throughout field work:

- Store data per classification requirements
- Maintain participant privacy
- Track data inventory
- Prepare for analysis phase

> Reference [UXR-002: Data Handling & Privacy](#sop/UXR-002/procedure) for all data management requirements.

## Phase Completion Checklist

Before moving to Phase 4, confirm:

- [ ] Target sample size achieved
- [ ] All sessions completed
- [ ] Consent obtained and documented for all participants
- [ ] Session recordings secured
- [ ] Notes completed for all sessions
- [ ] Data stored per compliance requirements
- [ ] Participant incentives processed
